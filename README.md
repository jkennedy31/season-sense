# Season Sense
SeasonSense is a modern weather dashboard designed to provide more than just the forecast. It aims to combine weather conditions with environmental information such as UV index, air quality, moon phases, tides, pollen, and other seasonal insights to help users better understand the world around them!

# Current Status
Version 1:
- Search for a city
- Display current weather
- Display temperature and conditions
